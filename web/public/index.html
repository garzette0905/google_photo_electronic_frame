표시할 사진 선택

  1. “구글 포토에서 선택” 클릭 또는 QR 스캔
  2. 위에 표시된 계정으로 로그인된 상태인지 확인 (폰에 여러 계정이 있으면 다른 계정을 고르면 선택이 완료되지 않습니다)
  3. 검색창에 인물 이름(예: 아버지) 또는 장소명 입력
  4. 원하는 사진들을 선택 후 완료
